# 10장 운영 우수성

## 모니터링

### 시스템 상태 모니터링

- 인프라 모니터링
  - cpu, 메모리, 네트워크, 디스크, 로드밸런서
- 애플리케이션 모니터링
  - 엔드포인트 호출, 응답 시간, 스로틀(용량이 부족해 처리되지 못한 요청 수), 오류
- 플랫폼 모니터링
  - 메모리 캐싱, DB, 컨테이너, 메시징 시스템, 검색 등
- 보안 모니터링
  - 네트워크 보안: 허가되지 않은 포트, 의심스러운 IP
  - 사용자 액세스: 허가되지 않은 사용자 액세스, 의심스러운 활동
  - 애플리케이션 보안: 바이러스 공격
  - 웹 보안: DDos, sql injection, XSS
  - 규정 준수
  - 데이터 보안: 허가되지 않은 데이터 액세스
